    Graphing tool

    The top rectangle is the input field.
    Equasions are entered as a js expression, with the only variable x. For example:

    x ** x + 3 - Math.sin(3 * x)
    If you begin with a bang (!), you can enter as just js. In this case, the input is expected to be a function:
    !conditional([ x => - x - 7, x => x < -5 ], [ x => -2 * x ** 2 - 12 * x - 12, _ => true ]);
    You can also enter this:
    !x => x * x

    You can only use ES6 arrow functions, so !x => x is allowed, but !function myFunc(x) { return x; } is not!
    Bottom arrows change the scale.
